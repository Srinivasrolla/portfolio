# 🚀 Rolla Srinivasa Rao — Personal Portfolio

A fully animated single-page portfolio website built with pure HTML, CSS & JavaScript.

🌐 **Live Site:** [https://Srinivasrolla.github.io/portfolio](https://Srinivasrolla.github.io/portfolio)

## ✨ Features
- Live particle network canvas animation
- Custom glowing cursor with trailing ring
- Shimmer gradient hero name
- 3D floating avatar card with ripple rings
- Scroll reveal animations on all sections
- Animated skill bars (triggered on scroll)
- Infinite tech stack marquee
- Project cards with gradient sweep on hover
- Fully responsive — works on mobile & desktop

## 🛠️ Tech Stack
Pure HTML · CSS Keyframes · Vanilla JavaScript  
Google Fonts: Syne + DM Sans

## 📁 Files
- `index.html` — The entire portfolio in one self-contained file
